🤖 App Check
Stops bots & scrapers from using your API
App Check makes Google verify every request comes from your real app (via invisible reCAPTCHA), so a stolen API key alone is useless to an attacker.
  1. Firebase Console → App Check → your web app → reCAPTCHA v3
  2. Register & copy the site key
  3. Paste it below & Save (then Enforce Firestore in App Check)
📜 Firestore Security Rules
The #1 thing protecting your data — set this once
Important: A Firebase API key is not a secret — it's safe to be public. What actually keeps strangers out of your data are these rules. They make Firestore reject anyone who isn't you, signed in with your Google account.
